# Gym Data Analysis Project

## Project Overview
This project analyzes gym and customer data using **Power BI** to provide insights into membership patterns, customer behavior, and business performance. The goal is to help gym management make data-driven decisions for improving membership retention, marketing strategies, and overall revenue.

## Features
- **Customer Analysis:** Track active and inactive members, age and gender distribution, and location-based trends.
- **Membership Insights:** Identify the most popular membership types and renewal patterns.
- **Attendance Trends:** Analyze peak hours and days of gym visits.
- **Revenue Analysis:** Visualize monthly payments, outstanding balances, and total income.
- **Interactive Dashboards:** Use filters and slicers to explore data dynamically.

## Dataset
The dataset contains multiple tables:
- `Customers`: Customer ID, Name, Age, Gender, City, Membership Type, Join Date, Status.
- `Payments`: Payment ID, Customer ID, Amount, Date, Payment Method.
- `Attendance`: Record of gym visits including Customer ID, Date, Duration, Activity Type.
- `Memberships`: Membership Type, Duration, Price.

> **Note:** The dataset has been anonymized to protect customer privacy.

## Tools
- **Power BI Desktop** for creating the dashboards and reports.
- Data cleaning and preparation done using Excel/Power Query.

## How to Use
1. Download the repository.
2. Open the `.pbix` file in Power BI Desktop.
3. Explore the dashboards using filters and slicers to analyze gym data.
4. Connect your own dataset by replacing the existing tables if needed.
